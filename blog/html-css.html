CSS Display

The purpose of this post is to discuss the difference between inline and inline-block in the display context.

The main difference between the two is that inline-block allows the developer to set a width and height on the element, as well as top and bottom padding & margins both being respected.

It is important to note that inline-block is different to display: block in that inline-block works horizontally and display: block stacks vertically. Display: block also inserts a linebreak after the element.
